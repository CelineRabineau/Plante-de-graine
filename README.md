# Plantes numériques

Plantes numériques est une installation créée pour mon projet de DNSEP.  
Elle présente plusieurs plantes virtuelles qui évoluent sur des cercles au sol, 
chacune générée et animée en P5.js.  
Parmi elles, Plante de Graine s’inspire de la dispersion naturelle des graines 
et montre comment une plante numérique peut continuer à « vivre » dans l’espace de l’installation, 
contrairement aux plantes réelles.
